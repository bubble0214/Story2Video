# 画布（Canvas）模块使用说明

## 概述

画布模块是一个基于 React Flow 的可视化故事创作工具，支持在节点画布上组织角色、场景、文本、图片、视频、音频等素材，并通过连线表达它们之间的关系。

**入口：** 左侧边栏 → 画布（`/canvas`）

---

## 界面布局

```
┌─────────────────────────────────────────────────────────────┐
│  工具栏（画布标题 + 快速添加 + 连线模式 + 保存）              │
├──────────┬──────────────────────────────┬───────────────────┤
│ 左侧面板  │        画布区域               │   节点属性面板     │
│ (192px)  │    (React Flow 画布)         │   (256px)         │
│          │                              │  点击节点后显示    │
│ 添加节点  │                              │                   │
│ · 角色    │   [角色]──→[场景]            │  · 名称编辑       │
│ · 场景    │    ↓                         │  · 内容编辑       │
│ · 文本    │   [文本]                     │  · 图片URL        │
│ · 图片    │                              │  · 关联任务       │
│ · 视频    │                              │  · 删除节点       │
│ · 音频    │                              │                   │
│          │                              │                   │
│ 素材列表  │  底部控制栏                    │                   │
│ (按类型)  │  [重置][整理][聚焦][缩放到]    │                   │
└──────────┴──────────────────────────────┴───────────────────┘
```

### 顶部工具栏

- **画布标题** — 点击可编辑，自动保存
- **快速添加按钮** — 文本 / 便签 / 图片
- **连线模式** — 切换开关，开启后拖拽节点连线更灵活
- **我的画布** — 查看/切换/删除所有已保存的画布
- **保存按钮** — 手动触发保存（画布每 5 秒自动保存）

### 左侧面板

- **添加节点** — 6 种节点类型：角色、场景、文本、图片、视频、音频
- **素材库** — 列出画布上所有已命名的角色和场景，点击可聚焦定位

### 画布区域

- **拖拽** — 移动节点
- **连线** — 从一个节点的底部拖到另一个节点的顶部
- **框选** — 按住 Shift 拖拽多选
- **滚轮** — 缩放画布
- **右键拖拽** — 平移视野

### 右侧节点属性面板

点击任意节点后在右侧打开，显示当前节点的属性编辑器：

| 节点类型 | 可编辑属性 |
|---------|-----------|
| 角色 | 名称、基础角色、外貌描述、描述文本、图片URL |
| 场景 | 名称、基础场景、外貌描述、描述文本、图片URL |
| 文本块 | 内容、关联项目 |
| 便签卡 | 内容、颜色 |
| 图片块 | 图片URL、关联项目 |
| 视频块 | 视频URL |
| 音频块 | 音频URL |

### 底部控制栏

- **重置视图** — 恢复默认视角
- **整理画布** — 按类型自动排列所有节点（角色/场景/文本等分组排列）
- **只看素材** — 仅显示角色和场景节点
- **小地图** — 切换右上角缩略图
- **网格对齐** — 开关吸附网格（15px）
- **缩放** — +/- 按钮和百分比显示

## 节点类型

| 类型 | 标识 | 用途 | 连线方式 |
|------|------|------|---------|
| **角色** (Character) | 带图片的方卡片 | 故事角色，可关联外貌设定 | 顶部入口/底部出口 |
| **场景** (Scene) | 16:9 图片卡片 | 故事场景/地点 | 顶部入口/底部出口 |
| **文本块** (TextBlock) | 文档图标 | 剧本内容、故事描述 | 顶部入口/底部出口 |
| **便签卡** (NoteCard) | 彩色卡片（默认黄） | 灵感记录、备注 | 顶部入口/底部出口 |
| **图片块** (ImageBlock) | 图片缩略图 | 参考图片、概念图 | 顶部入口/底部出口 |
| **视频块** (VideoBlock) | 播放器图标 | 视频素材 | 顶部入口/底部出口 |
| **音频块** (AudioBlock) | 音量图标 | 音频素材、配音 | 顶部入口/底部出口 |

## 高级功能

### 角色工具面板

选中角色节点后，底部会出现浮动工具栏：

- **三视图** — 生成角色的正面/侧面/背面视图（Sheet 面板）
- **灯光控制** — 调节光照角度、强度、色温（Sheet 面板）
- **相机控制** — 设置拍摄角度、景别、构图（Sheet 面板）
- **角色配置** — 完整角色设置对话框（包含语音参考）

### 场景工具面板

选中场景节点后，底部会出现浮动工具栏：

- **全景图** — 生成场景全景图（Sheet 面板）
- **相机控制** — 设置拍摄角度（Sheet 面板）

### 节点关联项目

文本块和图片块支持关联到一个已有的任务（Task）。在右侧面板的"关联项目"中选择任务后，节点上会显示关联标签。

### 素材视图

顶部的标签页可以切换到按类型浏览的素材视图（角色/场景/道具/材质），以卡片网格形式展示，支持快速定位到画布上的对应节点。

---

## 数据持久化

- **自动保存** — 画布每 5 秒自动保存（检测到变更时）
- **页面关闭保存** — 使用 `navigator.sendBeacon` 在页面关闭时保存
- **多画布支持** — 通过"我的画布"面板创建和切换多个画布
- **撤销历史** — 最多保留 50 步操作历史

## 常见操作流程

### 创建故事板

1. 点击左侧"角色"添加角色节点 → 填写角色名称和描述
2. 点击左侧"场景"添加场景节点 → 填写场景名称
3. 从角色节点底部拖拽连线到场景节点
4. 添加文本块记录该场景的剧情
5. 使用"整理画布"自动排列

### 关联项目素材

1. 添加图片块 → 在右侧面板粘贴图片 URL
2. 添加视频块 → 在右侧面板粘贴视频 URL
3. 添加音频块 → 在右侧面板粘贴音频 URL

### 角色设计

1. 选中角色节点 → 点击底部"Character Config"
2. 设置基础角色、外貌描述
3. 点击"Generate Three-View"生成三视图
4. 用灯光和相机控制面板调整展示效果
